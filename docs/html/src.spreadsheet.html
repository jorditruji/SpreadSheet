src.spreadsheet.spreadsheet module¶

class src.spreadsheet.spreadsheet.SpreadSheet¶

Bases: object

This is the main class that will contain all the Cells and Expressions making a SpreadSheet. All the functions contained in the menu, should be executed here.

Holds a maximum of 702 columns (from A to ZZ) even though it can be easily expanded
